

# 🎬 Cine-mate – Dynamic Movie Discovery Web Application

## 📌 Project Objective

Cine-mate was developed as a major frontend project to strengthen API integration skills and build a real-world React application inspired by modern streaming platforms like Netflix.

The project focuses on dynamic API handling using Axios, responsive UI design, and interactive user experience without backend integration.

---

## 🚀 Project Overview

Cine-mate is a Single Page Application (SPA) built using React and Vite that dynamically fetches movie data from an external movie API using Axios and displays it in a streaming-platform style interface.

This project was developed to practice:

* API integration using Axios
* Dynamic data rendering
* Client-side routing
* Modern UI structuring using Tailwind CSS

---

## ✨ Key Features

### 🎥 Dynamic Hero Section

* Displays currently trending movies
* Updates dynamically based on live API response
* Designed similar to OTT platforms

### 📂 Categorized Movie Sections

* Trending Movies
* Top Rated Movies
* Multiple categorized sections
* Horizontally scrollable movie rows

### ↔️ Smooth Horizontal Scrolling

* Left and right arrow navigation buttons
* Smooth scroll behavior for movie cards

### ❤️ Favorites Management

* Add movies to favorites
* Remove movies from favorites
* Managed using React state

### 📄 Detailed Movie Information Page

When a movie card is clicked, it displays:

* Movie overview and description
* Watch trailer link
* OTT platform availability
* Cast details
* Similar movie recommendations

### 🔄 API Integration with Axios

* Data fetched dynamically using Axios
* Asynchronous API calls
* Loading state handling
* Error handling implementation
